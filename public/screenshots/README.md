# Screenshots

Place screenshots here for the top page "画面で見るナゾトキ探偵団" section.

## File naming

| ID | Description | Recommended size |
|---|---|---|
| `session-progress.png` | Session wizard progress screen | 800x600 (4:3) |
| `evidence-cards.png` | Evidence card reveal screen | 800x600 (4:3) |
| `character-sheet.png` | Character sheet on mobile | 800x600 (4:3) |
| `voting-result.png` | Voting and reflection screen | 800x600 (4:3) |

## How to enable

After placing screenshots, edit `src/pages/index.astro` and change `hasImage: false` to `hasImage: true` for each item in the screenshots array.
